PASOS PARA ENTRAR EN HTML

los pasos para entar o crear la estructura nasica de html

Lista con viñetas

  • Crear carpeta en escritorio
  • Abrir dicha carpeta desde visual studio
  • en visual studio code crear archivo "index.html"
  • en linea 1 escribir html" seleccionar html:5"
  • escribir dentro de etiqueta BODY h1 (minuscula) y dar click al li que salga en las opciones de busqueda para escribir el titulo en el con letras grandes
  • escribir p (minuscula) para el subtitulo
  • escrbir li (minuscula) es cribir texto para la pagina
  • cambiar nombre de DOCUMENTO "mi primera pagina"
  • dentro de etiqueta BODY usar las etiquetas H1,H2,H3,H4,H5,H6,p,ul,ol,li
